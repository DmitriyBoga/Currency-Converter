Demo - https://dmitriyboga.github.io/Currency-Converter/ 
Приложение получает актуальные курсы валют через CurrencyAPI. Можно выбрать разные валюты и поменять их местами. Для удобства был создан "словарь" для перевода большинства валют на русский язык. 
Технологии: HTML, CSS, JavaScript, работа с API
